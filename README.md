# Landing Page - Curso de Desenvolvimento Web

Este projeto foi desenvolvido como atividade acadêmica para praticar conceitos básicos de HTML e CSS no desenvolvimento de uma landing page.

## Objetivo

O objetivo da atividade foi transformar um documento HTML em uma página visualmente organizada utilizando estilização com CSS, aplicando boas práticas de estrutura e organização de código.

## Tecnologias utilizadas

- HTML5
- CSS3

## Estrutura do projeto

O projeto está organizado da seguinte forma:

landing-page-curso-web
│
├── index.html
├── css
│ └── estilos.css
└── img
├── hero.png
└── aula.jpg


## Recursos aplicados

Durante o desenvolvimento foram utilizados:

- CSS externo
- CSS interno
- CSS inline
- Seletores CSS (elemento, classe, id)
- Pseudo-classe `:hover`
- Estilização de botões
- Organização visual de seções
